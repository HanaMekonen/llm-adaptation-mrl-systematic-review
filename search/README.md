# Search Strategy

This folder contains the complete search strategies used to identify studies for the systematic review:

**Techniques for Adapting Large Language Models to Low-Resource, Morphologically Rich Languages: A Systematic Review**

Because each bibliographic database supports different search syntax and indexing fields, the search strategy was adapted for each platform while preserving the same three conceptual blocks:

1. Large Language Models
2. Adaptation techniques
3. Low-resource and morphologically rich languages

The search was limited to publications between **January 2023 and March 2026**.

The files in this directory include:

- `scopus_query.txt` – Scopus search string
- `ieee_query.txt` – IEEE Xplore search string
- `sciencedirect_query.txt` – ScienceDirect search string
- `acl_query.txt` – ACL Anthology search strategy
- `search_log.md` – Record of database searches and retrieved studies
