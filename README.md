# Supplementary Materials for the Systematic Review

## Techniques for Adapting Large Language Models to Low-Resource, Morphologically Rich Languages: A Systematic Review

**Author:** Hana Mekonen  
**Supervisor:** Dr. Menore Tekeba  
**Institution:** Addis Ababa University, School of Electrical and Computer Engineering

---

## Overview

This repository contains the supplementary materials accompanying the systematic review:

> **Techniques for Adapting Large Language Models to Low-Resource, Morphologically Rich Languages: A Systematic Review**

The repository is provided to improve the transparency, reproducibility, and traceability of the review process. It includes the review protocol, database search strategies, screening records, data extraction sheets, quality assessment results, and supplementary figures used throughout the review.

---

## Repository Structure

| Folder | Description |
|---------|-------------|
| `protocol/` | Review protocol developed prior to conducting the literature search |
| `search/` | Complete search strategies adapted for each bibliographic database and repository |
| `screening/` | Study screening log, inclusion/exclusion decisions, and reasons for exclusion |
| `extraction/` | Data extraction template and extracted study characteristics |
| `quality/` | Quality assessment checklist and ratings for included studies |
| `figures/` | PRISMA 2020 flow diagram and supplementary figures |

---

## Reproducing the Review

Researchers wishing to reproduce this review may follow these steps:

1. Review the protocol in `protocol/`.
2. Execute the database search using the search strategies provided in `search/`.
3. Apply the eligibility criteria documented in the protocol.
4. Verify screening decisions using the records in `screening/`.
5. Inspect the extracted study data in `extraction/`.
6. Review the methodological quality assessments in `quality/`.
